# Wheel of Fortune

A simple, clean, and smooth Wheel of Fortune web application.

## How to use

1. Open `index.html` in your web browser.
2. Click the **SPIN** button to rotate the wheel.
3. The wheel will decelerate smoothly and land on a random name.
4. The winner will be displayed below the button.

## Customization

You can easily customize the names and colors in `script.js`:

```javascript
const names = [
    "Alice", "Bob", "Charlie", "David", 
    "Eve", "Frank", "Grace", "Heidi"
];
```

Enjoy!
